# Model Router Research

## Status

Draft research

## Objective

Determine how ai-manager should select an eligible model for a task and explain
that selection.

## Questions to Investigate

- Which task requirements influence routing?
- How are capability, quality, latency, cost, and quota balanced?
- Which constraints are mandatory and which are preferences?
- What fallback behavior is acceptable?
- How should routing decisions be recorded and explained?

## Candidate Inputs

- Task requirements.
- Model capabilities.
- Quota availability.
- Cost and latency policies.
- User or workflow preferences.

## Evaluation Criteria

- Correctness.
- Explainability.
- Predictability.
- Adaptability.
- Safe fallback behavior.

## Findings

To be researched.

## Decision Dependencies

Findings will inform routing policy, quota interfaces, and observability
requirements.
