# Prompt Builder Research

## Status

Draft research

## Objective

Determine how ai-manager should define, compose, version, validate, and evaluate
prompts.

## Questions to Investigate

- Which prompt parts should be reusable?
- How are variables typed and validated?
- How are model-specific adaptations represented?
- How are prompt versions linked to workflow outcomes?
- Which evaluation methods are required before promotion?

## Candidate Capabilities

- Reusable prompt definitions.
- Structured variables and validation.
- Version history.
- Model-specific rendering.
- Evaluation metadata.

## Evaluation Criteria

- Reproducibility.
- Reviewability.
- Composability.
- Traceability.
- Model portability.
